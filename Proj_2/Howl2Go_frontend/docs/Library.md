# Howl2Go Frontend Library Documentation

## Overview
Howl2Go uses a modern and performance-optimized frontend stack.  
All integrations follow official configurations with minimal adjustments to ensure consistency, maintainability, and reliability.

---

## Libraries & Tools

| Library / Tool         | Version | Purpose / Integration Method                                   |
|-----------------------|---------|---------------------------------------------------------------|
| Next.js               | 15      | Main app framework and routing; pages are located in `/app/`  |
| React                 | 19      | Core UI library for building reusable components               |
| TypeScript            | 5       | Adds static typing for safer and more maintainable code        |
| Tailwind CSS          | 4       | Utility-first CSS framework for styling                        |
| Framer Motion         | Latest  | Handles animations and transitions                             |
| Lucide React Icons    | Latest  | Provides vector icons for the UI                               |
| Jest                  | Latest  | Unit and component testing framework                           |
| React Testing Library | Latest  | Testing library for React components                           |
